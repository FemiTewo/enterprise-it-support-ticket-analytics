# Enterprise IT Support Ticket & SLA Analytics Dashboard

## Project Overview

This project analyzes IT support ticket data to simulate an enterprise application support and IT operations environment.

The goal is to identify ticket trends, SLA breaches, resolution-time patterns, operational bottlenecks, and high-risk support categories. The project demonstrates practical skills in Python, SQL, Power BI, data cleaning, SLA analysis, and business reporting for IT operations and enterprise support teams.

## Business Problem

Enterprise support teams often manage large volumes of incidents across multiple applications, regions, and support levels. Without proper analytics, managers may struggle to identify recurring issues, SLA breaches, delayed resolutions, and high-risk operational areas.

This project provides a structured analytics approach to help IT operations leaders monitor service performance and make data-driven improvement decisions.

## Project Objectives

- Analyze IT support ticket volume and trends
- Measure SLA compliance and breach patterns
- Identify high-risk issue categories and support areas
- Evaluate average ticket resolution time
- Track operational bottlenecks by priority, status, and support category
- Build an executive dashboard for IT service performance monitoring
